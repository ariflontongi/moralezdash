What is the reason for the file restrictions?

Locked files are defined as those that can only be read and written to by one application or process at a given time. Locked files are used by all operating systems. In most circumstances, locking a file serves the aim of ensuring that it cannot be modified, transferred, or destroyed while it is being used, either by you or by another computer process.
